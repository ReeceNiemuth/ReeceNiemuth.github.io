CompTIA Security+ is a foundational cybersecurity certification that validates baseline security skills. It's often required for DoD 8570 compliance.

---

## Exam Overview (SY0-701)

| Detail | Information |
|--------|-------------|
| **Domains** | 5 |
| **Questions** | 90 (multiple choice + PBQs) |
| **Duration** | 90 minutes |
| **Passing Score** | 750/900 |
| **DoD 8570** | IAT Level II, IAM Level I |

---

## The 5 Domains

1. **General Security Concepts** (12%)
2. **Threats, Vulnerabilities, and Mitigations** (22%)
3. **Security Architecture** (18%)
4. **Security Operations** (28%)
5. **Security Program Management and Oversight** (20%)

## Tips for Success

1. **Know the acronyms** - Security+ is acronym-heavy
2. **Understand attack types** - Know how attacks work, not just names
3. **Practice PBQs** - Performance-based questions can be tricky
4. **Use the exam objectives** - Study directly from CompTIA's list
