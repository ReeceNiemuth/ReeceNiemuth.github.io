The OSCP (Offensive Security Certified Professional) is a hands-on penetration testing certification that requires compromising machines in a lab environment.

---

## Exam Overview

| Detail | Information |
|--------|-------------|
| **Format** | Hands-on practical exam |
| **Duration** | 23 hours 45 minutes + 24 hours reporting |
| **Passing Score** | 70 points |
| **Style** | Compromise machines in a lab network |

---

## Key Topics

- Information Gathering & Enumeration
- Buffer Overflows
- Web Application Attacks
- Active Directory Attacks
- Privilege Escalation (Linux & Windows)
- Client-Side Attacks
- Tunneling and Pivoting
- Metasploit Framework

## My Methodology

```
1. Enumerate thoroughly (nmap, gobuster, etc.)
2. Document everything
3. Try the obvious first
4. Don't rabbit-hole - move on and come back
5. Take breaks during the exam
```
